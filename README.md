# Проекты 4-9: Место 
https://innadobrynina.github.io/mesto/index.html

Страница создана в рамках прохождения 4-9 спринтов курса Веб-разработчика Яндекс.Практикума. В работе применены  HTML, CSS, JavaScript (Асинхронный JS, ООП, API), проверка валидности данных форм, git, Webpack.
### Возможности

Добавление и удаление фотографии
"Лайк" для фотографии
Редактирование профиля пользователя
Данные хранятся на сервере и загружаются с сервера
Валидация форм на стороне клиента
